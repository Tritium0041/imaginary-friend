# 时空拍卖行 (Chronos Auction House)

## 游戏概述

《时空拍卖行》是一款竞拍策略游戏，3-5名玩家扮演穿越时空的古董商。每局游戏进行5轮，每轮包含多个阶段。

玩家通过竞拍获得不同时代的文物，操纵时代倍率，并巧妙使用功能卡来获取最高胜利点数。

## 玩家人数

3-5人

## 资源系统

### 玩家资源
- **资金 (money)**: 初始 20 金币，最小值 0。用于竞拍和交易。
- **胜利点数 (victory_points)**: 初始 0。游戏结束时点数最高者获胜。

### 全局资源
- **时空稳定性 (stability)**: 初始 100，最小值 0，最大值 200。每次拍卖成交会扣减稳定性。降为 0 时游戏提前结束。

## 时代与稀有度分类

### 时代 (era)
文物分为三个时代，每个时代有独立的倍率（影响文物最终价值）：
- **古代** — 初始倍率 ×1.0（范围 ×0.5 ~ ×2.5）
- **近代** — 初始倍率 ×1.0（范围 ×0.5 ~ ×2.5）
- **未来** — 初始倍率 ×1.0（范围 ×0.5 ~ ×2.5）

### 稀有度 (rarity)
- **传奇** — 最珍贵
- **稀有** — 较珍贵
- **常见** — 基础文物

## 游戏区域

- **拍卖区 (auction_pool)**: 展示待拍卖的文物
- **系统仓库 (system_warehouse)**: 存放出售给系统的文物
- **文物弃牌堆 (discard_pile)**: 弃置的文物
- **功能卡弃牌堆 (card_discard_pile)**: 使用过的功能卡
- **事件弃牌堆 (event_discard_pile)**: 已处理的事件卡

## 游戏流程

每轮按以下阶段顺序进行：

### 1. 挖掘阶段
- 从文物牌库补充拍卖区至「玩家数 + 1」件文物
- 每 3 回合，每位玩家从功能卡库抽 1 张功能卡

### 2. 拍卖阶段
- 对拍卖区中的每件文物进行拍卖
- 拍卖类型由文物决定：
  - **公开拍卖**: 玩家依次出价，pass 后不可重入，最后一位出价者获胜
  - **密封竞标**: 所有玩家同时秘密出价，最高出价者获胜。平局时按行动顺序裁定
- 拍卖成交后：
  - 赢家支付出价金额
  - 文物加入赢家收藏
  - 时空稳定性扣减（扣减值 = 文物的 time_cost）

### 3. 交易阶段
- 玩家间可自由交易文物和资金
- 玩家可将文物出售给系统仓库

### 4. 回购拍卖阶段
- 系统仓库中的文物进行公开拍卖

### 5. 事件阶段
- 翻开事件区的事件卡
- 对事件进行投票
- 执行得票最高的事件

### 6. 投票阶段
- 起始玩家发起时代倍率调整提议
- 所有玩家用投票标记（✅/❌）投票
- 投票标记使用后不可恢复

### 7. 稳定阶段
- 玩家可选择弃置功能卡或捐献资金来修复时空稳定性

## 胜利条件

### 计分公式
```
总分 = Σ(文物基础价值 × 该时代倍率) + (剩余资金 / 10) + 套装加成
```

### 游戏结束条件
- 文物牌库为空且当前拍卖阶段结束
- 时空稳定性降为 0

### 套装加成
- **时空博学家**: 拥有 3 个不同时代各至少 1 件文物 → +5 VP
- **时代专精者**: 同一时代 3 件文物 → +5 VP
- **关键词收集者**: 同一关键词 3 件文物 → +5 VP
